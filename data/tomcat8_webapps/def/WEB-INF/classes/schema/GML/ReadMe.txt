This archive contains schema documents for the EPSG Geodetic Parameter Registry v1.0, last updated 2012-06-20.

It replaces the v0.1 schemas of 2008-03-20 with effect from August 2012.
See Guidance Note 7 part 3, Annex E, for a summary of changes between v0.1 and v1.0.

The application schemas for the EPSG Registry conform to GML version 3.2.1.

The content of the archive is:
- Folder "epsg":   the EPSG Registry schema.
- Folder "gml":    the GML 3.2.1 schema set; the root document of the GML schema is file "gml.xsd".
- Folder "iso":    the imported GMD schema and related schemas (see ISO/TS 19139:2007).
- Folder "w3C":    the imported W3C schemas (XLink 1.1, etc).
